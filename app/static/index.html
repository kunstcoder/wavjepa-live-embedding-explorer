Audio Foundation Model Explorer

WavJEPA와 Audio-JEPA 임베딩 비교

같은 오디오를 waveform 기반 WavJEPA와 spectrogram 기반 Audio-JEPA로 임베딩하고, mean pooled embedding을 PCA 또는 t-SNE로 축소해 시각화합니다.

Model: WavJEPA / Audio-JEPA
Input: 16kHz / 32kHz Mono
Output: 768-d pooled vector

분석 설정

첫 추론 시 필요한 모델 파일을 `models/` 아래로 다운로드합니다.

오디오 파일을 드래그하거나 클릭 WAV, FLAC, OGG, MP3 등 libsndfile이 읽을 수 있는 형식

Compare는 같은 파일을 두 모델로 각각 임베딩합니다.

배치 업로드 분석에 사용됩니다.

Realtime Microphone

실시간 모드는 안정성을 위해 PCA로 고정됩니다. 마이크 입력을 짧은 청크로 잘라 선택한 모델의 trajectory를 2D/3D 공간에 그립니다. 다른 PC에서 마이크를 쓰려면 HTTPS 또는 localhost가 필요합니다.

-45 dBFS · RMS 0.0056 값을 높일수록 더 큰 소리만 통과합니다.

너무 작은 청크는 임베딩과 trajectory에 추가하지 않습니다.
