Audio Foundation Model Explorer

WavJEPA 임베딩을 2D/3D 공간으로 펼쳐보기

`labhamlet/wavjepa-base`를 로컬에 내려받아 오디오 특징을 추출하고, mean pooled embedding을 PCA 또는 t-SNE로 축소해 시각화합니다.

Model: WavJEPA Base
Input: 16kHz Mono
Output: 768-d pooled vector

분석 설정

첫 추론 시 모델 스냅샷을 `models/` 아래로 다운로드합니다.

오디오 파일을 드래그하거나 클릭 WAV, FLAC, OGG, MP3 등 libsndfile이 읽을 수 있는 형식

배치 업로드 분석에 사용됩니다.

Realtime Microphone

실시간 모드는 안정성을 위해 PCA로 고정됩니다. 마이크 입력을 짧은 청크로 잘라 누적 trajectory를 2D/3D 공간에 그립니다. 다른 PC에서 마이크를 쓰려면 HTTPS 또는 localhost가 필요합니다.

-45 dBFS · RMS 0.0056 값을 높일수록 더 큰 소리만 통과합니다.
